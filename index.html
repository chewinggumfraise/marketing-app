Le Flux de Travail Idéal

📥

1. La Demande

Une nouvelle demande de projet est créée (interne ou client).

👤

2. Centralisation

Toute demande est envoyée à Guillaume.

Décision : Urgence?

Guillaume évalue la priorité.

🎨

3. Brief: Stacy (Graphisme)

Délai: 2 jours ouvrables

💻

3. Brief: Jo-anny (Marketing)

Délai: 2 jours ouvrables

🔄

4. Révision Interne

Le travail complété retourne à Guillaume pour une première validation.

👑

5. Approbation Finale

La version finale est envoyée à Dominic pour le dernier mot.

Nos Règles d'Or

  • ✓
    Un Projet = Un Fil de Courriel: Fini les communications éparpillées. Tout reste au même endroit.
  • ✓
    Brief Complet Dès le Début: Toute demande doit inclure: Type (Print/Web), Dimensions, Textes, Images voulues, etc.
  • ✓
    Guillaume Toujours en Boucle: Il doit être en CC sur les communications importantes du projet.
  • ✓
    Respect des Délais: Minimum 2 jours ouvrables pour chaque projet.
  • ✓
    Les Urgences Passent par Guillaume: Lui seul peut approuver un projet "rush" et ajuster les priorités.
